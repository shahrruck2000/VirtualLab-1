Load Test on Single Phase Tranformer

Documentation

Circuit Explanation
Instructions
  1. To Enable the ON button set the fixing ratio and KVA rating of the transformer using dropdowns
  2. To Enable the DPSTS button switch on the power supply(Uisng ON button>
  3. To Enable the primary rheostat turn on the DPSTS(CheckButton on primary side).
  4. To enable the load variac set the rated voltage on primary side using rheostat(slider on primary side).
  5. To save the reading click the save the data button
  6. After completing the experiment click the Go to table button to view table
Tabulation-Calculations
Instructions
  1. Click the Generate button to generate the tables with saved values
  2. Enter the calculated Efficency and Regulation value in respective column
  3. To cross check you calculated value with system click the Check button
  4. If your values are correct you will receive a congratulation message or else you will get a warning message
